## 核心作用

本仓库用于自动同步 Tailscale DERP Map，并生成 OpenClash / Clash 可用规则集。

作用说明：

- DERP = Tailscale 中继服务器
- 无法直连时会走 DERP
- Clash 误代理会导致：
  -> 流量二次转发
  -> 延迟增加

本规则强制：

✔ DERP 流量 DIRECT  
✔ 避免代理嵌套  
✔ 降低组网延迟

## OpenClash rule 类型说明

classical = DOMAIN + IPCIDR 混合匹配（推荐）
ipcidr = 仅IP（不完整）
domain = 仅域名（不完整）

结论：
必须使用 classical，否则会漏流量
